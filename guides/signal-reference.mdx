description: Every field in the scores object.

For routing decisions, use `tier_verdicts`. The fields below support logging, reviewer interfaces, and certification records.

## Scores

| Field | Range | Description |
|-------|-------|-------------|
| `confidence_score` | 0–1 | Overall AI probability. 0 = human, 1 = AI. Use for displaying confidence to reviewers |
| `geometry_score` | 0–1 | Multi-signal corroborating score, stable across audio processing variations |
| `fusion_score` | 0–1 | Combined geometric fusion of all detection signals |
| `prototype_score` | 0–1 | Drives the press-safe tier. Include in certification records |
| `spectral_score` | 0–1 | Primary spectral analysis signal |
| `frequency_artifact_score` | 0–1 | Detects frequency-domain generation artifacts |
| `temporal_consistency_score` | 0–1 | Temporal pattern consistency across the track |
| `perceptual_score` | 0–1 | Perceptual model analysis |
| `phase_coherence_score` | 0–1 | Phase-domain coherence analysis |
| `representation_score` | 0–1 | Embedding-space representation analysis |
| `hf_energy_score` | 0–1 | High-frequency energy distribution analysis |
| `codec_artifact_score` | 0–1 | Audio codec artifact detection |
| `temporal_stability` | float | Acoustic consistency across the track. Available for logging |
| `reliability` | 0–1 | Confidence in the result based on audio quality and duration. Below `0.6` — treat with extra caution |

## Origin attribution

| Field | Description |
|-------|-------------|
| `origin` | Predicted generator: `"suno"`, `"udio"`, `"elevenlabs"`, `"human"`, `"unknown"` |
| `origin_confidence` | Confidence in the prediction (0–1) |

Origin attribution is supplemental — use it for logging and analytics. It does not affect `verdict` or `tier_verdicts`.

## Audit fields

| Field | Description |
|-------|-------------|
| `calibration_version` | Version of the model calibration that produced this result. Include in certification records |
| `processed_at` | ISO timestamp of when analysis completed |
