---
title: Get Job
description: Retrieve the status and progress of a job.
---

## Endpoint

```http
GET /jobs/{job_id}
```

## Example request

```bash
curl https://app.jobsbyhumans.com/bulk/jobs/job_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

```json
{
  "id": "job_abc123",
  "name": "Q1 Catalog Scan",
  "status": "processing",
  "total": 100,
  "processed": 47,
  "succeeded": 45,
  "failed": 2,
  "verdict_counts": {
    "human": 38,
    "suspicious": 7
  },
  "created_at": "2026-02-27T10:00:00Z",
  "completed_at": null
}
```

## Job status values

| Status | Description |
|--------|-------------|
| `queued` | Job received, waiting to start |
| `processing` | Tracks are being analyzed |
| `complete` | All tracks finished |
| `failed` | Job-level failure (e.g. bad manifest URL) |
