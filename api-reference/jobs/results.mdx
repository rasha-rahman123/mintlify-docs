---
title: Get Job Results
description: Retrieve per-track results for a completed job.
---

## Endpoint

```http
GET /jobs/{job_id}/results
```

Results are returned as **newline-delimited JSON (NDJSON)** — one record per line. This format is efficient for streaming large result sets.

## Example request

<CodeGroup>
```bash cURL
curl https://app.jobsbyhumans.com/api/v1/jobs/job_abc123/results \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests, os

response = requests.get(
    "https://app.jobsbyhumans.com/api/v1/jobs/job_abc123/results",
    headers={"Authorization": f"Bearer {os.environ['HS_API_KEY']}"},
    stream=True,
)

for line in response.iter_lines():
    if line:
        record = json.loads(line)
        print(record["item_id"], record["verdict"])
```

```typescript TypeScript
const response = await fetch(
  "https://app.jobsbyhumans.com/api/v1/jobs/job_abc123/results",
  { headers: { "Authorization": `Bearer ${process.env.HS_API_KEY}` } }
);

const text = await response.text();
const records = text.trim().split("\n").map(line => JSON.parse(line));

for (const record of records) {
  console.log(record.item_id, record.verdict);
}
```
</CodeGroup>

## Response (NDJSON)

```
{"item_id":"track-001","status":"success","verdict":"human","tier_verdicts":{"press_safe":"human","human_safe":"human","recall":"human"},"scores":{"confidence_score":0.12,"geometry_score":0.18,"reliability":0.91},...}
{"item_id":"track-002","status":"success","verdict":"suspicious","tier_verdicts":{"press_safe":null,"human_safe":"suspicious","recall":"suspicious"},"scores":{"confidence_score":0.79,"geometry_score":0.71,"reliability":0.88},...}
{"item_id":"track-003","status":"failed","verdict":null,"error":"audio_load_failed"}
```

Each record matches the full result schema. See [Reading Results](/guides/reading-results) for field descriptions.

## Filtering

| Query param | Values | Description |
|------------|--------|-------------|
| `verdict` | `human`, `suspicious` | Filter by verdict |
| `status` | `success`, `failed` | Filter by processing status |
| `page` | integer | Page number (default: 1) |
| `per_page` | integer | Results per page (default: 500, max: 1000) |

```bash
# Only return suspicious tracks
curl "https://app.jobsbyhumans.com/api/v1/jobs/job_abc123/results?verdict=suspicious" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
