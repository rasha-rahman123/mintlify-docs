---
title: Get Track
description: Retrieve a previously analyzed track by ID.
---

## Endpoint

```http
GET /tracks/{track_id}
```

## Example request

```bash
curl https://app.jobsbyhumans.com/api/v1/tracks/trk_xyz789 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

Returns the same result schema as [Analyze Track](/api-reference/tracks/analyze).

```json
{

  "status": "success",
  "verdict": "human",

    "press_safe": "human",
    "human_safe": "human",
    "recall": "human"
  },
  "scores": {
    "confidence_score": 0.08,
    "geometry_score": 0.12,

    "reliability": 0.94
```
