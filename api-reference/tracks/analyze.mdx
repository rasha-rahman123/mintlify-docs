---
title: Analyze Track
description: Analyze a single audio track synchronously.
---

## Endpoint

```http
POST /analyze
```

## Request body

```json
{
  "url": "https://your-cdn.com/track.mp3"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `url` | string | Yes* | Publicly accessible audio URL |
| `file_id` | string | Yes* | File identifier for signed URL resolution |

*Provide either `url` or `file_id`, not both.

## Query parameters

| Param | Values | Description |
|-------|--------|-------------|
| `detail` | `full`, `simple` | Response detail level. Default: `full` |

Pass `detail=simple` for a reduced ~10-field response. See [Simplified Responses](/guides/simplified-responses).

## Example request

<CodeGroup>
```bash cURL
curl -X POST https://app.jobsbyhumans.com/api/analyze \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://cdn.example.com/track.mp3"}'
```

```python Python
import requests, os

result = requests.post(
    "https://app.jobsbyhumans.com/api/analyze",
    headers={"Authorization": f"Bearer {os.environ['HS_API_KEY']}"},
    json={"url": "https://cdn.example.com/track.mp3"}
).json()
```

```typescript TypeScript
const result = await fetch("https://app.jobsbyhumans.com/api/analyze", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.HS_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({ url: "https://cdn.example.com/track.mp3" }),
}).then(r => r.json());
```
</CodeGroup>
